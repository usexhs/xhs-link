xhslink Resolver

Removes hidden tracking parameter in xhslink dot com.

STILL UNDER EXTENSIVE DEVELOPMENT

This site is also available at xhslink.icu

Usage

Suppose you have a link like: (http(s)://)xhslink.com/YOUR_SHORT_CODE

Redirect

Replace "com" with "icu" in you link.

https://xhslink.icu/YOUR_SHORT_CODE

Resolve the code

https://xhs.li/code/YOUR_SHORT_CODE

Resolve the full URL

https://xhs.li/full/?url=(http(s)://)xhslink.com/YOUR_SHORT_CODE
